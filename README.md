# Active Directory Home Lab

This project documents my process of building a Windows Server 2019 Active Directory environment in a virtual lab.

The goal of this lab is to gain hands-on experience with enterprise identity management, domain administration, and Windows infrastructure.

---

## Lab Environment

Hypervisor
VirtualBox

Domain Controller
Windows Server 2019

Workstation
Windows 11

Domain Name
laceylab.local

---

## Skills Practiced

Active Directory Domain Services (AD DS)
DNS configuration
Static IP configuration
Domain controller deployment
User and group management
Organizational Units
Domain joining workstations
Basic Group Policy
